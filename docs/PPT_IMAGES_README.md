## 使用方法

### 在 PowerPoint 中使用

1. **插入圖片**：
   - 開啟 PowerPoint
   - 選擇「插入」→「圖片」→「此裝置」
   - 選擇 `docs/ppt_images/` 目錄中的 `*_ppt.png` 檔案

2. **調整大小**：
   - 圖片已經是 16:9 比例，可以直接填滿投影片
   - 建議使用「填滿」模式以獲得最佳效果

3. **建議佈局**：
   - 可以將圖片作為投影片背景
   - 或使用「圖片置中」佈局
   - 圖片內容已居中，周圍有白色背景

## 技術細節

- **原始內容**: 完全保留，未做任何修改
- **縮放方式**: 保持原始寬高比，使用高品質 LANCZOS 重採樣
- **背景處理**: 白色背景 (#FFFFFF)，圖片居中顯示
- **檔案大小**: 已優化，適合簡報使用

## 重新生成

如果需要重新生成，執行：

```bash
python convert_to_ppt_format.py
```

## 注意事項

- 垂直方向的圖片（如 diagram_02）在 16:9 格式中會顯示得較小，這是為了保持原始比例
- 如需更大的顯示，可以考慮：
  1. 在 PowerPoint 中手動調整大小
  2. 或使用「圖片填滿」模式
  3. 或將圖片分成多張投影片
