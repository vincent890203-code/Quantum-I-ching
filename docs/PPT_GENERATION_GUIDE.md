# PowerPoint 架構圖生成指南

## 目標

生成適合 PowerPoint 16:9 格式的架構圖：
- **尺寸**: 1920x1080 像素
- **字體**: 最小 28 號字體
- **內容**: 簡化，只保留核心資訊
- **佈局**: 充分利用 16:9 寬螢幕空間

## 使用 Playwright 生成高品質圖片（推薦）

### 安裝步驟

```bash
# 1. 安裝 Playwright
pip install playwright

# 2. 安裝 Chromium 瀏覽器
playwright install chromium
```

### 生成高解析度圖片

```bash
python generate_ppt_diagrams_highres.py
```

這會生成：
- 1920x1080 高解析度圖片
- 字體大小優化（28-32 號）
- 充分利用 16:9 空間

## 使用 Mermaid Live Editor（最簡單）

如果 Playwright 安裝有問題，可以使用線上工具：

1. **開啟**: https://mermaid.live/
2. **複製**: 從 `ARCHITECTURE_DIAGRAM_PPT.md` 複製 Mermaid 程式碼
3. **貼上**: 貼到編輯器左側
4. **調整**: 在編輯器中可以調整字體大小和佈局
5. **下載**: 點擊 "Actions" → "Download PNG"
6. **調整尺寸**: 使用圖片編輯軟體調整為 1920x1080

## 在 PowerPoint 中使用

1. **插入圖片**: 插入 → 圖片 → 此裝置
2. **選擇檔案**: 選擇 `docs/ppt_images/` 中的 `ppt_*.png`
3. **調整大小**: 圖片已經是 16:9，可以直接填滿投影片
4. **建議**: 使用「圖片填滿」模式以獲得最佳效果

## 注意事項

- Mermaid 的字體大小主要取決於節點大小
- 簡化文字內容可以讓節點更大，字體更清晰
- 如果字體仍然太小，建議使用 Mermaid Live Editor 手動調整
- 或使用圖片編輯軟體後處理（調整對比度、銳化等）
