# Quantum I-Ching 股市卜卦系統 - 15分鐘簡報架構

- 專案資訊：日期、作者/團隊
- 視覺：易經卦象圖示 + 金融圖表元素

## 第2張：問題背景
**時間：約1分鐘**

**內容：**

  - 結合量化分析與傳統智慧

  - 提供結構化的投資建議

## 第5張：資料編碼邏輯
**時間：約1分30秒**

**內容：**
- **Whale Volume 四象邏輯**
  - 將價格與成交量轉成四象（6/7/8/9）
  - 6（老陰）：下跌 + 異常大量
  - 7（少陽）：上漲 + 正常量
  - 8（少陰）：下跌 + 正常量
  - 9（老陽）：上漲 + 異常大量
- **六爻卦象生成**
  - 滾動窗口（6天）組成六爻
  - 儀式數字序列 → 二進制編碼 → 64卦查詢
- **之卦計算**
  - 動爻（6/9）會變動，產生未來卦

**視覺：編碼流程圖 + 四象對照表**

---

## 第6張：之卦策略與貞/悔架構
**時間：約1分30秒**

**內容：**
- **傳統之卦解法**
  - 依動爻數量（0-6）動態選擇查詢策略
  - 0動爻：全盤接受（本卦卦辭）
  - 1動爻：焦點在單一動爻
  - 2動爻：主客對照（下爻貞/上爻悔）
  - 3動爻：對沖時刻（本卦貞/之卦悔）
  - 4-5動爻：趨勢反轉（之卦貞/本卦悔）
  - 6動爻：極端反轉
- **貞/悔架構**
  - **貞 (Zhen)**：主體、支撐、長期、進場、持有
  - **悔 (Hui)**：客體、阻力、短期、出場、風險
- **金融映射**
  - 貞 → 趨勢支撐、主要方向、可倚賴的層級
  - 悔 → 風險管理、壓力位、需警惕的層級

**視覺：動爻策略決策樹 + 貞/悔對照表**

## 第7張：RAG知識庫系統
**時間：約1分鐘**

**內容：**
- **知識庫來源**
  - 從 [john-walks-slow/open-iching](https://github.com/john-walks-slow/open-iching) 下載
  - 64卦完整資料（主卦 + 六爻）
  - 約450份文件（64主卦 + 386爻）
- **向量化處理**
  - ChromaDB 本地持久化
  - SentenceTransformers (all-MiniLM-L6-v2) 嵌入
  - 支援語義搜尋與精確檢索
- **檢索策略**
  - 依之卦策略決定查詢範圍
  - 嚴格對應卦象ID與爻位
  - 確保易經原文與市場卦象100%一致

**視覺：知識庫結構圖 + 檢索流程**

## 第8張：AI解讀系統
**時間：約1分鐘**

## 第9張：Web儀表板功能
**時間：約1分鐘**

**內容：**
- **Streamlit Dashboard 特色**
  - 左側：輸入股票代號、問題、市場類型選擇
  - 中間：最近60日K線圖（Plotly White主題）
    - 技術註解：MA20（黃色，貞/Support）、MA60（紫色，悔/Resistance）
  - 右側：I-Ching市場卦象視覺化
    - 無動爻：單一本卦
    - 有動爻：本卦 → 之卦並排顯示
    - 動爻橙色高亮標示
    - **情緒儀表**：自訂進度條顯示多方能量（紅色=多方，綠色=空方）
  - **量化橋接指標列**（圖表與解讀之間）：
    - 收盤價（含漲跌幅）
    - RVOL（相對成交量）
    - 系統狀態（依動爻數量：Stable/Active/Volatile）
    - 趨勢強度（Bullish 🐂 / Bearish 🐻）
  - **Oracle AI解讀**（帶邊框容器）：
    - 🚀 投資快訊（永遠顯示）
    - 🎯 關鍵操作建議（永遠顯示，依語氣自動選擇色彩）
    - 📜 易經原文（可摺疊）
    - 💡 現代解讀（可摺疊）
- **多市場支援**
  - 台股（TW）：自動補 `.TW` 後綴，顯示中文公司名稱
  - 美股（US）：直接使用代碼
  - 加密貨幣（CRYPTO）：自動補 `-USD`
- **卦象一致性**
  - Calculate Once, Use Everywhere
  - 畫面顯示與AI解讀使用相同卦象

**視覺：Dashboard截圖 + 功能標註（包含量化橋接、情緒儀表、資訊層級）**
